#概述
本文基于网友共享的繁体中文油猴插件优化而来，支持简体中文，行文中不仅仅是将繁体改为中文，也替换了很多简体中文和繁体中文用词不一样的地方，使得简体中文版更适合简体中文用户的文字习惯。

#使用
1. 安装Tampermonkey插件
2. 下载脚本文件，并导入Tampermonkey，或者直接引用脚本路径
3. 在需要使用的网页上点击Tampermonkey图标，选择“启用脚本”
4. 如果需要，可以点击Tampermonkey图标，选择“禁用脚本”
